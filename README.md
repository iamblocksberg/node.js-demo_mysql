# Node.js - Demo MySQL

*Using MySQL with Node.js*

### Feature
- Add Data
- Show Data
- Delete Data

### How to Install
1. Import [file Database](https://github.com/iamblocksberg/node.js-demo_mysql/blob/master/_File%20Database/awesome_db.sql) to your Database 
1. cd project
1. npm install
  1. npm start
