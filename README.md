# Node.js - Demo MySQL

*Using MySQL with Node.js*

### Feature
- Add Data
- Show Data
- Delete Data

### How to install
* Import [file Database](https://github.com/iamblocksberg/node.js-demo_mysql/blob/master/_File%20Database/awesome_db.sql) to your Database 
* cd project
* npm install
* npm start
